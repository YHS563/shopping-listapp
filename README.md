# 🛒 쇼핑 리스트 앱

바닐라 JavaScript로 만든 간단한 쇼핑 리스트 웹 앱입니다.

## 기능

- **아이템 추가** — 입력창에 텍스트 입력 후 버튼 클릭 또는 Enter 키
- **완료 체크** — 체크박스 클릭으로 완료/미완료 토글
- **아이템 삭제** — 개별 항목 우측 ✕ 버튼으로 삭제
- **필터** — 전체 / 미완료 / 완료 항목 필터링
- **완료 항목 일괄 삭제** — 완료된 항목 한 번에 제거
- **데이터 유지** — localStorage에 저장되어 새로고침 후에도 유지

## 실행 방법

별도의 설치 없이 `shopping-list.html` 파일을 브라우저에서 열면 바로 실행됩니다.

## 자동화 테스트

[Playwright](https://playwright.dev/)를 사용한 E2E 테스트가 포함되어 있습니다.

```bash
# 패키지 설치
npm install

# 로컬 서버 실행 (8787 포트)
python -m http.server 8787

# 테스트 실행 (새 터미널)
node test-shopping.mjs
```

### 테스트 항목 (총 20개)

| 테스트 | 내용 |
|--------|------|
| 아이템 추가 (버튼) | 버튼 클릭으로 아이템 추가 |
| 아이템 추가 (Enter) | Enter 키로 아이템 추가 |
| 빈 입력 무시 | 공백 입력 시 추가되지 않음 |
| 체크 기능 | 완료 상태로 변경 및 요약 반영 |
| 체크 해제 | 완료 → 미완료로 복귀 |
| 필터 - 미완료 | 미완료 항목만 표시 |
| 필터 - 완료 | 완료 항목만 표시 |
| 필터 - 전체 | 모든 항목 표시 |
| 개별 삭제 | 특정 아이템 삭제 |
| 완료 일괄 삭제 | 완료된 항목 전체 삭제 |
| localStorage 저장 | 새로고침 후 데이터 유지 |

## 기술 스택

- HTML / CSS / JavaScript (바닐라)
- localStorage
- Playwright (테스트)
